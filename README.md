# 🎓 LMS Learn — Learning Management System

A **full-stack Learning Management System (LMS)** built using the **MERN Stack** that allows instructors to create and sell courses while students can purchase and watch them with a structured learning experience.

The platform includes **course creation, video streaming, payments, authentication, and progress tracking**.

---

# 🌐 Live Demo

**Live Application:**  
https://learning-management-system-sunny.onrender.com/

---

# 🚀 Features

## 👨‍🏫 Instructor Features
- Create and manage courses
- Upload course lectures (video)
- Bulk upload videos
- Edit course content
- View instructor dashboard
- Track students and revenue
- Replace or delete lectures

---

## 👨‍🎓 Student Features
- Browse available courses
- Purchase courses
- Watch course lectures
- Track learning progress
- Continue from the last watched lecture
- Course completion celebration
- Rewatch completed courses

---

## 🔐 Authentication
- User registration & login
- JWT based authentication
- Role-based access (Instructor / Student)
- Protected routes

---

## 💳 Payment Integration
- PayPal payment gateway
- Order creation and verification
- Secure course access after purchase

---

## 📹 Video Streaming
- Cloudinary video hosting
- Video player with custom controls
- Free preview lectures
- Lecture progress tracking
